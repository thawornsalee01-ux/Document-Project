# Document Version Compare 

ลดระยะเวลาการอ่านเพื่อ **หาข้อแตกต่างหรือสิ่งที่เปลี่ยนแปลงของเอกสารเวอร์ชั่นใหม่**.

## Document Version Compare คืออะไร?

Document Version Compare คือ เว็บแอพพลิเคชั่นสำหรับ **การเปรียบเทียบเอกสารและสรุปการเปลี่ยนแปลงด้วย AI** 🤖 .

### **แล้วสามารถทำอะไรได้บ้าง?** 
- สามารถเปรียบเทียบไฟล์ .pdf กับ .pdf และ .docx กับ .docx ได้
- สรุปการเปลี่ยนแปลงได้ละเอียดพร้อมทั้งบอกผลกระทบของการเปลี่ยนแปลง
- วิเคราะห์ความเสี่ยงของการเเก้ไขของเอกสารแต่ละเวอร์ชั่นได้และแสดงออกมาเป็นกราฟพร้อมทั้งเหตุผลของความเสี่ยง
- ย้อนดูประวัติการใช้งานการเปรียบเทียบเอกสาร
- AI Chatbot สำหรับการถามคำถามเพิ่มเติมของแต่ละการเปลี่ยนแปลง
- สามารถเปรียบเทียบกับเอกสารเดิมในระบบได้

### **ทำงานอย่างไร?**
1. นำเข้าข้อมูลของเอกสารทั้ง 2 ฉบับ
2. แบ่งย่อหน้าของแต่ละเอกสาร
3. นำเเต่ละย่อหน้าของเอกสารไป embedding 
4. นำ vector ที่ได้มาจับคู่ similarity เพื่อที่จะเปลี่ยนเทียบการแก้ไขแต่ละย่อหน้า
5. แยกย่อหน้าที่ไม่มีการเปลี่ยนแปลงออก
6. นำย่อหน้าที่มีการเปลี่ยนแปลงส่งเข้าโมเดล LLM เพิ่อที่จะสรุปหัวข้อและเนื้อหาของการเปลี่ยนแปลง
7. นำผลของโมเดล LLM จากข้อ 6 ส่งให้โมเดล LLM อีกโมเดลเพิ่อให้แนะนำจากการเปลี่ยนแปลง
8. นำผลจากโมเดล LLM ทั้งข้อ 6 และ 7 มาสรุปผลด้วยโมเดล LLM ตัวสุดท้ายเพื่อสรุปการเปลี่ยนแปลทั้งหมด

### **เหมาะกับใคร?**
- องค์กรภาครัฐเนื่องจากมีแก้ไขเอกสารราชการหลาย version หรือ ตรวจสอบการเปลี่ยนแปลงของเนื้อหา เพื่อลดระยะเวลาการอ่านเอกสารหลายหน้า ซึ่งระบบจะช่วยแสดงให้เห็นความแตกต่างของแต่ละย่อหน้าและสรุปการเปลี่ยนแปลงทั้งหมด เพื่อลดความผิดพลาดของการอ่าน
- บริษัทเอกชนที่ทำงานเกี่ยวกับกฎหมายหรือทำสัญญาทางธุรกิจเนื่องจากเอกสารมักจะมี Draft, Reveiw, Final Version


### ทำไมถึงต้องใช้?

- **การรักษาความปลอดภัยของข้อมูล** เนื่องจากเอกสารต่างๆของภายในองค์กรส่วนมากมักเป็นข้อมูลสำคัญที่มีความละเอียดอ่อน หากนำเอกสารเหล่านี้ไปใช้งานผ่านแอปพลิเคชันภายนอก อาจเกิดความเสี่ยงที่ข้อมูลจะถูกเข้าถึงโดยบุคคลที่ไม่ได้รับอนุญาต หรืออาจเกิดการรั่วไหลของข้อมูลได้การใช้ระบบภายในองค์กรช่วยให้สามารถควบคุมการเข้าถึงข้อมูลได้อย่างมีประสิทธิภาพ และลดความเสี่ยงด้านความปลอดภัยของข้อมูล
- **การปกป้องความเป็นส่วนตัวของข้อมูล** เอกสารบางประเภทอาจมีข้อมูลส่วนบุคคล เช่น ข้อมูลพนักงาน, ข้อมูลลูกค้า, ข้อมูลทางธุรกิจ การนำเอกสารดังกล่าวไปประมวลผลผ่านระบบภายนอก อาจทำให้ข้อมูลถูกจัดเก็บหรือประมวลผลบนเซิร์ฟเวอร์ของผู้ให้บริการ ซึ่งองค์กรไม่สามารถควบคุมได้อย่างเต็มที่ ดังนั้น การใช้ระบบภายในองค์กรจะช่วยให้ข้อมูลทั้งหมดถูกจัดเก็บและประมวลผลภายในระบบขององค์กรเอง ส่งผลให้สามารถรักษาความเป็นส่วนตัวของข้อมูลได้ดียิ่งขึ้น

## จุดเด่นของDocument Version Compare
- **การเปรียบเทียบเอกสารอัตโนมัติ (Automatic Document Comparison)** ระบบสามารถเปรียบเทียบเนื้อหาระหว่างเอกสารสองเวอร์ชัน เช่น Draft และ Final Version ได้โดยอัตโนมัติ พร้อมทั้งแสดงความแตกต่างของข้อความที่ถูกเพิ่ม ลบ หรือแก้ไขอย่างชัดเจน ทำให้ผู้ใช้งานสามารถตรวจสอบการเปลี่ยนแปลงของเอกสารได้อย่างรวดเร็วและแม่นยำ
- **การแสดงผลความแตกต่างอย่างชัดเจน (Visual Difference Highlight)** ระบบจะแสดงความแตกต่างของเอกสารในรูปแบบที่เข้าใจง่าย เช่น การไฮไลต์ข้อความที่มีการเปลี่ยนแปลง ซึ่งช่วยให้ผู้ใช้งานสามารถมองเห็นจุดที่ถูกแก้ไขได้ทันที ลดเวลาในการตรวจสอบเอกสารด้วยตนเอง
- **การวิเคราะห์การเปลี่ยนแปลงด้วย AI (AI-based Analysis)** ระบบสามารถใช้เทคโนโลยี AI ในการวิเคราะห์การเปลี่ยนแปลงของเอกสาร เพื่อช่วยระบุจุดที่มีความสำคัญหรืออาจมีผลกระทบต่อเนื้อหาของเอกสาร ทำให้ผู้ใช้งานสามารถตัดสินใจได้ง่ายขึ้น
- **การจัดเก็บประวัติการเปรียบเทียบ (Comparison History)** ระบบสามารถบันทึกประวัติการเปรียบเทียบเอกสารในแต่ละครั้ง ทำให้ผู้ใช้งานสามารถย้อนกลับไปตรวจสอบการเปรียบเทียบที่ผ่านมาได้ และช่วยในการติดตามการแก้ไขเอกสารในแต่ละเวอร์ชัน
- **การรักษาความปลอดภัยของข้อมูล (Secure Internal System)** ระบบถูกออกแบบให้ใช้งานภายในองค์กร ทำให้เอกสารทั้งหมดถูกจัดเก็บและประมวลผลภายในระบบขององค์กร ช่วยลดความเสี่ยงในการรั่วไหลของข้อมูล และเพิ่มความปลอดภัยในการจัดการเอกสารสำคัญ
