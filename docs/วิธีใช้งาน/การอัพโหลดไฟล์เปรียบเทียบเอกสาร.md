# การอัพโหลดไฟล์เปรียบเทียบเอกสาร

การอัพโหลดไฟล์เปรียบเทียบเอกสารให้เข้ามาหน้า **Compare** ที่อยู่ทางด้านซ้ายจะได้หน้าต่างดังนี้

## ตัวอย่างหน้าการอัพโหลดเอกสาร

![Upload Document](/img/compare_page.png)

- อัพโหลดไฟล์เวอร์ชั่นแรกไว้ที่ช่อง**ด้านซ้าย**และอัพโหลดไฟล์เวอร์ชั่นถัดไปที่ช่อง**ด้านขวา**
- รองรับไฟล์ **.pdf** และ **.docx** ไม่สามารถอัพโหลดประเภทไฟล์ข้ามเวอร์ชั่นกันได้เช่น pdf เปรียบเทียบกับ docx  **แนะนำใช้ pdf เปรียบเทียบกันเพื่อประสิทธิภาพที่ดีที่สุด**
- สามารถตั้งชื่อเอกสารเเละเวอร์ชั่นได้จากปุ่มตั้งชื่อเอกสารด้านบนปุ่มอัพโหลดไฟล์**แนะนำควรตั้งชื่อเอกสารและเวอร์ชั่นเพื่อสำหรับการกลับมาค้นหาข้อมูลย้อนหลังได้ง่าย** ดังรูปภาพ
